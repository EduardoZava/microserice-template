# API Gateway (Traefik)

Traefik v3 API Gateway for routing requests to microservices.

## Features
- Dynamic routing via Docker labels
- CORS middleware
- Rate limiting
- Secure headers
- Dashboard at http://localhost:8080

## Configuration
- Static config: `traefik.yml`
- Dynamic config: `dynamic/routes.yml`

## Dashboard
Access Traefik dashboard at: http://localhost:8080/dashboard/
